This project simulates the motion of a marble falling through a viscous liquid and collects data at regular time intervals. The goal is to estimate the $\ast$$\ast$friction rate (\$\textbackslash{}alpha\$)$\ast$$\ast$ using the recorded trajectory data.

The project consists of\+:
\begin{DoxyItemize}
\item A {\bfseries{friction module}} that calculates the friction rate from velocity data.
\item A {\bfseries{test model}} that generates synthetic trajectory data.
\item A {\bfseries{data analysis pipeline}} that estimates the friction rate based on the collected data.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md2}{}\doxysection{Physical Model}\label{md_README_autotoc_md2}
